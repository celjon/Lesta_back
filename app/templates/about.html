Что такое TF-IDF?

TF-IDF (Term Frequency-Inverse Document Frequency) — это статистическая мера, используемая для оценки важности слова в контексте документа, являющегося частью коллекции документов.

Компоненты TF-IDF:

1. TF (Term Frequency) - Частота термина

Показывает, как часто определенное слово встречается в документе. Чем выше частота, тем важнее слово для этого документа.

Формула: TF = (Количество вхождений слова в документе) / (Общее количество слов в документе)

2. IDF (Inverse Document Frequency) - Обратная частота документа

Показывает, насколько уникально или редко слово в коллекции документов. Слова, которые редко встречаются в коллекции, имеют высокий IDF.

Формула: IDF = log(Общее количество документов / Количество документов, содержащих слово)

3. TF-IDF

Является произведением TF и IDF и служит числовой статистикой, отражающей важность слова для документа в контексте коллекции.

Формула: TF-IDF = TF × IDF

Практическое применение TF-IDF:

  • Поисковые системы — для ранжирования релевантности документа к запросу
  • Классификация текстов — для выделения ключевых особенностей документов
  • Автоматическое реферирование — для выбора наиболее значимых предложений
  • Рекомендательные системы — для анализа сходства документов

О нашем анализаторе:

Этот инструмент позволяет загрузить текстовый файл и выполнить анализ TF-IDF для слов в нем. Анализатор предварительно обрабатывает текст, удаляя стоп-слова и знаки пунктуации, затем рассчитывает значения TF, IDF и TF-IDF для каждого слова.

Результаты сортируются по убыванию значения IDF, показывая наиболее уникальные и информативные слова в тексте.
